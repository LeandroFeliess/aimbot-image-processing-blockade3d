# Blockade 3D Classic - Installatie Info

## Installatie Locatie
```
C:\Program Files (x86)\Steam\steamapps\common\BLOCKADE
```

## Belangrijke Bestanden

### Executables
- **Blockade.exe** (653 KB) - Hoofd executable
- **BlockadeEAC.exe** (3.9 MB) - Easy Anti-Cheat launcher
- **BlockadeLauncher.exe** (441 KB) - Game launcher
- **GameAssembly.dll** (77 MB) - Unity game assembly

### Anti-Cheat
- **EasyAntiCheat** folder - EAC installatie
- **BlockadeEAC.exe** - EAC process

### Game Data
- **Blockade_Data** folder - Unity game data
- **AssetBundles** folder - Game assets

## Belangrijke Informatie

1. **Easy Anti-Cheat (EAC)**: 
   - Het spel gebruikt EAC voor anti-cheat
   - EAC draait als apart proces (BlockadeEAC.exe)
   - Dit is waarom we ultra-safe mode nodig hebben

2. **Unity Engine**:
   - GameAssembly.dll = Unity IL2CPP build
   - Blockade_Data = Unity game data
   - Window title kan variëren

3. **Window Detection**:
   - Window title kan zijn: "Blockade", "Blockade 3D Classic", of "Blockade 3D"
   - De window finder moet alle varianten ondersteunen

## Aanbevelingen voor Aimbot

1. **Window Title**: 
   - Huidige detectie werkt (vindt "Blockade")
   - Maar misschien moet het ook "Blockade 3D Classic" zoeken

2. **EAC Bypass**:
   - Alle anti-cheat maatregelen zijn al geïmplementeerd
   - External image processing (geen memory injection)
   - Human-like input patterns

3. **Testing**:
   - Test eerst in menu (geen EAC actief)
   - Test dan in-game (EAC actief)
   - Monitor voor bans/waarschuwingen
