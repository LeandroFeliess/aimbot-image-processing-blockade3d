## Stap-voor-Stap Uitleg

### Stap 1: Open Command Prompt (Terminal)

**Methode A: Via Verkenner**
1. Open **Verkenner** (Windows Explorer)
2. Ga naar de map: `C:\Users\Gebruiker\aimbot-image-processing-blockade3d`
3. Klik in de adresbalk en type: `cmd` en druk op Enter
4. Een zwart venster (Command Prompt) opent zich

**Methode B: Via Start Menu**
1. Druk op **Windows toets** (of klik Start)
2. Type: `cmd` of `Command Prompt`
3. Klik op "Command Prompt"
4. Type: `cd C:\Users\Gebruiker\aimbot-image-processing-blockade3d`
5. Druk op Enter

**Methode C: Via PowerShell**
1. Druk op **Windows toets + X**
2. Kies "Windows PowerShell" of "Terminal"
3. Type: `cd C:\Users\Gebruiker\aimbot-image-processing-blockade3d`
4. Druk op Enter

---

### Stap 2: Start De Aimbot

In het zwarte venster (Command Prompt/PowerShell) type je:

```bash
python main_074_3.6.5.py
```

**Druk op Enter**

## â“ Problemen?

### "python is not recognized"
**Oplossing:**
- Gebruik: `python -m pip --version` om te testen
- Of probeer: `py main_074_3.6.5.py` (met `py` in plaats van `python`)

### "No module named..."
**Oplossing:**
- Installeer ontbrekende packages: `python -m pip install -r requirements.txt`

### Script start niet
**Oplossing:**
- Check of je in de juiste map bent: `cd C:\Users\Gebruiker\aimbot-image-processing-blockade3d`
- Check of het bestand bestaat: `dir main_074_3.6.5.py`
